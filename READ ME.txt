 YouTube to MP3 Python Script

This Python script allows you to download YouTube videos and convert them into MP3 files using 'yt-dlp' and 'moviepy'. You can easily integrate it into your project or use it as a standalone tool.

 Requirements

Before you start, make sure you have the following installed:

- Python 3.6+ (preferred)
- **Python libraries**:
  - python-dotenv
  - yt-dlp
  - moviepy
  Visit this link to download ffmpeg
  -https://www.gyan.dev/ffmpeg/builds/#git-master-builds

You can install all required libraries by running the following commands:

pip install python-dotenv
pip install yt-dlp
pip install moviepy
